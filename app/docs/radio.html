Welcome Web UI ( Settings MP3 Radio Walkie-Talkie Expert )

The radio settings allows to define up to 8 preset stations with their frequency and name (the name is only useful in the Web UI mode). Manual tuning via turning the control button can be disabled if desired and for better reception with headphones, the radio can be forced to use mono, which improves the signal. By default manual tuning is enabled and stereo is used when headphones are plugged in.
