Welcome Web UI ( Settings Playlists Radio Walkie-Talkie Expert )

To enter the Web UI mode, long press both the control and volume buttons. The player will enable WiFi and search for available known networks to connect to. If none is found, it will start a new network, which unless specified otherwise is set to 

    SSID: mp3-player
    PASS: mp3-player

To display the Web UI, enter mp3-player.local in your browser (or click this link if already connected). The Web UI will appear at the Settings page.

The UI consists of different sections (selectable from the header at the top of the page), each section comprises of various panels. A panel has a header and may start in a collapsed mode, in which case there is a button in the right corner of the header, clicking it opens the pabel:
