Welcome Web UI ( Settings MP3 Radio Walkie-Talkie Expert )

These are the expert settings for the player. It is not recommended to use them unless you really know what you are doing as you can brick the player when used wrong. But don'ty worry, inserting a clean SD card will always reset everything to factory defaults.

The first panel offers shortcuts for various commands: The player can be put to sleep immediately, time and walkie talkie synchronization can be enforced, or the player can be reset (keeps data, but looses time). The Format command replaces all non-music files on the SD card with their default values, so is pretty destructive. All network connections, radio stations, playlist names and walkie-talkie settings will be most, but no music files should be deleted.
